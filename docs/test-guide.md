## Test Guide

To run the tests for this project, you can use the following command:

```
pnpm run test
```

### Testing with demo app

To test the library with the demo app, you can run:

#### Build the tact-js library

```
pnpm run build
```

#### Update demo app dependencies

```
pnpm install
```

#### Run the demo app

```
pnpm run dev
```
